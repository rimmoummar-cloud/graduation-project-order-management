Graduation Project – Legacy Version

📌 Overview

This project was developed as part of my graduation work using Visual Basic (.NET).
It represents one of my early full systems where I implemented core application logic and database operations.

The system includes:

User authentication (basic)
Order management
CRUD operations (Insert, Update, Delete)

---

⚙️ Features

Basic authentication system
Order creation and management
Database operations (Insert / Update / Delete)
Multi-form desktop application

---

⚠️ Important Notes

This project reflects my earlier stage in software development, and it contains several limitations:

The authentication system is very basic and not secure
The UI/Design is limited due to the nature of Windows Forms
Some database operations are not fully optimized or error-safe
The overall architecture does not follow best practices I use today

🚀 What I Learned

While working on this project, I gained experience in:

Building a complete system from scratch

🔄 How I Would Improve It Today

▶️ How to Run

To run this project locally:

Open the solution in Visual Studio
Configure the database connection (update connection string)
Make sure the database is created and connected
Run the application

---

📎 Notes

The database is not included by default.
You may need to create your own database or import a provided SQL file (if available).
